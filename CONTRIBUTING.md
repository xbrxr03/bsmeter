# Contributing to BS Meter

Thanks for your interest! This is a hackathon project (SLOP SCAN, May 29 – Jun 1, 2026) so things move fast.

## How to Contribute

1. Fork the repo
2. Create a feature branch (`git checkout -b feature/your-feature`)
3. Make your changes
4. Run tests (`npm test`)
5. Commit and push
6. Open a PR

## Code Ownership

See [docs/ARCHITECTURE.md](docs/ARCHITECTURE.md) for the full file structure and who owns what.

## Detection Approach

BS Meter measures information **quality**, not AI-ness. See [docs/ARCHITECTURE.md](docs/ARCHITECTURE.md) for the signal stack and scoring methodology.

## What We Don't Do

- Count em-dashes or punctuation
- Wrap GPTZero/Originality.ai
- Ask an LLM "is this AI?"
- Shame people for using AI
